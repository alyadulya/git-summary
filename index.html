Intro

Git adalah salah satu version control system yang berguna untuk mencatat setiap perubahan file pada source code kita. Git bermanfaat untuk menyimpan seluruh versi source code, mempermudah berkolaborasi membangun sistem, berkontribusi di proyek open source, memudahkan tracking perubahan, memahami cara deploy modern, dsb. Beberapa layanan hosting yang menggunakan Git antara lain Bitbucket, Github, dan Gitlab.
